###  DIN UPPGIFT: Besvara följande fråga i denna md-fil / 3 poäng

Beskriv hur man kan sätta upp Typescript i ett Reactprojekt. Redogör för fördelar respektive nackdelar att använda Typescript i React.

För att sätta upp TypeScript i ett React-projekt kan vi använda Create React App med TypeScript-mallen eller manuellt konfigurera TypeScript i vårt projekt genom att installera TypeScript-paketet och skapa en tsconfig.json-fil.

Fördelar med med att använda typescript är samma som har TypeScript på sig själv mankan hitta felen inan kompilering så kan vi se tack vare definierade typer vilka problem eller fel kan vi ha. Läsbarhet är enklare och det gör att koden kan vara fixad snabbare.
IDE-stöd är en annan fördel för att många utvecklingsmiljöer erbjuder bättre stöd för kodkomplettering, refaktoriseringar och felhantering med TypeScript.

Nackdelen är att det är svårare för de som inte är bekanta med Typescript och kan vara problematisk i början för att man måste lära sig nya koncept och att jobba med typer, interface och andra funktioner som ren Javascript inte har. Inlärningskurva kan vara högt på det sätt. Kompilerings tider är längre också i typescript för undersökning att problem.

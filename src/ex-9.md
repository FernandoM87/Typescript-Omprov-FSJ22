Beskriv vad generics är i TypeScript och varför de är användbara.
Hur skiljer sig generics från att använda "any" i TypeScript?

Generics i TypeScript tillåter utvecklare att skapa återanvändbara komponenter, funktioner eller klasser som kan arbeta med olika typer av data, samtidigt som de behåller typsäkerheten. Genom att använda generics kan kod skrivas som är flexibel och kan hantera olika datatyper utan att förlora fördelarna med typesäkerhet.

Skillnaden mellan generics och att anvanda "any" är att generics tillåter typsäkerhet  medan "any" tvinger Typescript att ignorera typkontroll för den variabeln eller uttrycket. Generics bevarar typsäkerheten genom att lämna att utvecklare specifiera vilka typer som skaanvändas vid körning. Any tillåter att jobba med vilken variabel som helstutan typningsproblem eller fel.
